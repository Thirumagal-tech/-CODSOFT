# Projects Included

## 1. Movie Genre Classification

### Overview

The Movie Genre Classification project aims to automatically predict the genre of a movie based on its description or plot summary. This helps streaming platforms and recommendation systems categorize movies efficiently.

### Technologies Used

* Python
* Scikit-learn
* Pandas
* NumPy
* TF-IDF Vectorization

### Machine Learning Algorithms

* Naive Bayes
* Logistic Regression

### Workflow

1. Data collection and preprocessing
2. Text cleaning and tokenization
3. Feature extraction using TF-IDF Vectorizer
4. Model training using classification algorithms
5. Model evaluation and prediction

### Applications

* Movie recommendation systems
* Automatic movie categorization
* Content filtering systems

# 2. Customer Churn Prediction

### Overview

Customer Churn Prediction identifies customers who are likely to stop using a subscription-based service. Businesses can use this model to improve customer retention strategies and reduce revenue loss.

### Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib / Seaborn

### Machine Learning Algorithms

* Logistic Regression
* Random Forest
* Gradient Boosting

### Workflow

1. Data preprocessing and cleaning
2. Feature selection (usage behavior, demographics)
3. Model training using multiple algorithms
4. Performance evaluation using accuracy and confusion matrix
5. Predicting potential churn customers

### Applications

* Telecom companies
* Subscription platforms
* Customer retention strategies

# 3. SMS Spam Detection

### Overview

This project builds a machine learning model that classifies SMS messages as **Spam or Ham (Not Spam)**. It helps filter unwanted messages and improve communication security.

### Technologies Used

* Python
* Scikit-learn
* Pandas
* Natural Language Processing (NLP)

### Machine Learning Algorithms

* Naive Bayes
* Logistic Regression

### Workflow

1. Data preprocessing and text cleaning
2. Feature extraction using TF-IDF Vectorizer
3. Model training and testing
4. Evaluation of classification performance
5. Spam prediction for new messages

### Applications

* Email and SMS filtering systems
* Communication security
* Fraud message detection
